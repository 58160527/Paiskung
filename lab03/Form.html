Form in Bootstrap

Post by Admin, Feb 10, 2017.
Form Bootstrap

วันนี้เราจะมาพูดถึงเรื่องการใช้งาน Form ของ Bootstrap ซึ่งก็ไม่ได้มีอะไรซับซ้อนมากมาย โดยเราแค่เรียกใช้ class “form-control” ใน textbox combo-box หรือ textarea แค่นี้ input ต่างๆ ก็แสดงผลในรูปแบบของ bootstrap โดย input ต่างๆ ที่มีการเรียกใช้ class “form-control” จะมีลักษณะที่โค้งมน และมีความกว้าง 100% ใน tag ที่ input นี้อยู่

Inline form

ก็ตามชื่อครับ เป็น from ที่มีพวก input อยู่ในแถวเดียวกัน ซึ่งหน้าตาก็จะประมาณนี้

Horizontal form

ตัวนี้จะเป็นฟอร์มตามแนวนอนครับ โดย input ต่างๆ ปกติ จะอยู่คนละแถวกันครับ จะมีอยู่แถวเดียวกันก็เพียง label เท่านั้น ซึ่ง Horizontal form นั้นมีการนำ grid system ที่เราได้กล่าวไปแล้วใน บทความที่แล้ว เรื่อง contanier และ grid ครับ ซึ่งหน้าตาของ Horizontal form ก็จะประมาณนี้ครับ
